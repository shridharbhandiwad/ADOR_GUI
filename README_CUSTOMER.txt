================================================================================
                     RADAR VISUALIZATION APPLICATION
                            Version 1.0.0
================================================================================

QUICK START
-----------
1. Extract this ZIP file to any folder
2. Double-click "RadarVisualization.exe" to run
3. No installation required!


SYSTEM REQUIREMENTS
-------------------
- Windows 10 or Windows 11 (64-bit)
- No Qt installation needed (all libraries included)
- Visual C++ Redistributable may be required (see below)


FEATURES
--------
- Semi-circular PPI radar display (-90 deg to +90 deg)
- Real-time FFT spectrum analysis
- Target tracking table with all parameters
- UDP data reception on configurable port
- Built-in data simulation for testing
- 50ms update rate for real-time performance
- Modern light theme UI


USAGE
-----
1. Launch the application
2. Configure UDP settings if receiving data from external source
3. Or enable Simulation mode to test with sample data
4. Adjust range and display settings as needed


TROUBLESHOOTING
---------------
If the application doesn't start:

1. Missing DLL Error:
   Install Visual C++ Redistributable from Microsoft:
   https://aka.ms/vs/17/release/vc_redist.x64.exe

2. Application crashes:
   - Run from command prompt to see error messages
   - Check Windows Event Viewer for details

3. Firewall Warning:
   Allow the application through Windows Firewall for UDP reception
